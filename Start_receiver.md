# Как начать принимать сообщения на ценник по сети LoRa

В текущей реализации существует "шлюз", который по сети LoRa отправляет на ценники новые данные, которые должны быть отображены

Ценник должен быть создан согласно схеме и подключен к пк или иметь элементы питания.

После подключения выполняется для обнаружения имени порта выполняется команда:
```
[System.IO.Ports.SerialPort]::getportnames()
```
P.S.: команда актуальна для powershell (windows). В другой ОС команда может отличаться


После необходимо загрузить на микроконтроллер файлы:
```
lora_receiver_display.py
pico_display.py
```

Это можно сделать командами:
```
py -3.10 -m mpremote connect com10 cp .\to_controller\lora_receiver_display.py :main.py
py -3.10 -m mpremote connect com10 cp .\to_controller\pico_display.py :pico_display.py
```
com10 - может отличаться, узнается командой "...getportnames"

Далее перезагружаем контроллер и запускаем
```
py -3.10 -m mpremote connect com10 reset
py -3.10 -m mpremote connect com10 ls
```

Должны увидеть вывод 
```
>>>
```

Если этого нет, нажмите Enter





После успешной загрузки файлов на микроконтроллеров и подключения к нему необходимо выполнить:
```
>>> import main
>>> main.main()
```

Теперь устройство готово к приему сообщений.
Осталось только отправить сообщение
