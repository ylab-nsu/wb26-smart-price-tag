Описание системы
Цель проекта:

Создание системы управления автономными электронными ценниками на базе ESP32 микроконтроллеров для отображения информации о товарах.

Основные возможности:
  • Управление названиями товаров
  • Управление ценами в реальном времени
  • Отображение веса товаров
  • Мониторинг состояния батареи устройств
  • Прямая связь с устройствами ESP32
Технические особенности:
  • ESP32 с дисплеями
  • WiFi подключение к локальной сети
  • Автономная работа от аккумуляторов
  • Минимальный набор данных для эффективной работы
